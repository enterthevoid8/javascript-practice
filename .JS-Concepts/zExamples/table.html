HTML Table Example

Student Information
ID Name Age
1 John Doe 20
2 Jane Smith 22
Total Students 2

Explanation:

  • <table>: Defines the start of the table.
  • <caption>: Provides a title or explanation for the table.
  • <thead>: Contains the header row(s) of the table.
  • <tbody>: Contains the body (data rows) of the table.
  • <tfoot>: Contains the footer row(s) of the table.
  • <tr>: Defines a table row.
  • <th>: Defines a header cell (in the header row).
  • <td>: Defines a data cell.
  • colspan: Specifies the number of columns a cell should span.
  • border-collapse: Specifies the table border-collapse behavior.
